UDP:
	est protocole sans connexion
TCP:
	est protocole avec connexion

Ethernet, Wifi : 
	protocoles de la liaison de données

IP:
	protocoles de la couche réseau


> Couche physique :
	- Assurer la transmission des bits sur les moyens physiques d’interconnexion (cable coaxial, fibre, microondes, etc.).

> Couche liaison de données :
	- Contrôle d’erreurs 
	- Contrôle de flux ( capacité des trames )
	- Gestion de lien
	- La synchronisation ( Mécanisme qui permet la reconnaissance du début et de fin de trame )

> Couche Reseaux :
	- Fragmentation (par le routeur)
	- Routage ( but : trouver le meilleur chemin pour acheminer le paquet )
